# CRAFT – Construction Reports & Analytics for Flawless Tasks

Современная система управления строительными дефектами с разграничением ролей и аналитикой в реальном времени.

## Описание

CRAFT — это веб-приложение для отслеживания и управления дефектами на строительных объектах. Система позволяет эффективно контролировать процесс выявления, устранения и приемки строительных недостатков с учетом ролей пользователей.

## Основные возможности

### Управление дефектами
- Создание и регистрация дефектов с детальным описанием
- Категоризация по типам (строительство, отделка, инженерия, безопасность, документация)
- Установка приоритетов (критический, высокий, средний, низкий)
- Назначение исполнителей и установка сроков устранения
- Отслеживание статусов (новый, в работе, на проверке, закрыт, отменен)
- Прикрепление фотографий дефектов
- История изменений и комментариев

### Работа с проектами
- Управление несколькими строительными объектами
- Просмотр статистики по каждому проекту
- Фильтрация и поиск дефектов
- Экспорт данных в CSV

### Аналитика и отчетность
- Дашборд с ключевыми метриками
- Визуализация данных через графики и диаграммы
- Отслеживание просроченных дефектов
- Экспорт отчетов в CSV:
  - Сводный отчет по всем проектам
  - Детальный отчет по проектам
  - Полный список дефектов

### Система ролей

#### Инженер
- Создание новых дефектов
- Просмотр назначенных задач
- Обновление статуса работ
- Добавление комментариев

#### Менеджер проекта
- Все возможности инженера
- Назначение исполнителей
- Изменение приоритетов и сроков
- Перевод дефектов на проверку
- Закрытие устраненных дефектов

#### Директор
- Просмотр аналитики по всем проектам
- Доступ ко всем дефектам
- Экспорт отчетов в CSV
- Контроль общей эффективности

## Технологический стек

- **Frontend**: React 18 с TypeScript
- **UI**: Tailwind CSS
- **Иконки**: Lucide React
- **Графики**: Recharts, Chart.js
- **Уведомления**: React Hot Toast
- **Сборка**: Vite
- **База данных**: LocalStorage (демо-режим)

## Установка и запуск

### Предварительные требования
- Node.js версии 18 или выше
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут находиться в директории `dist/`

### Предварительный просмотр продакшен-сборки

```bash
npm run preview
```

## Основные функции

### Создание дефекта
1. Войдите как инженер или менеджер
2. Выберите проект
3. Нажмите "Создать дефект"
4. Заполните форму с описанием и деталями
5. Опционально прикрепите фото

### Работа с дефектом
- **Назначение**: менеджер выбирает исполнителя
- **Выполнение**: инженер переводит в статус "В работе"
- **Проверка**: после устранения отправляется на проверку
- **Закрытие**: менеджер проверяет и закрывает дефект

### Экспорт отчетов
1. Войдите как директор
2. На главной странице нажмите одну из кнопок экспорта:
   - **Сводный отчет** — общая статистика
   - **Проекты** — детали по каждому проекту
   - **Дефекты** — полный список дефектов
3. CSV файл автоматически загрузится
